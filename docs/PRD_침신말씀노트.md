# PRD: 침신 말씀노트 (Chimshin Bible Note)
## 제품 요구사항 문서 v1.0

---

## 1. 제품 개요

### 1.1 제품명
- **국문**: 침신 말씀노트
- **영문**: Chimshin Bible Note
- **부제**: AI 말씀비서와 함께하는 신앙 성장

### 1.2 핵심 가치 제안
> "침례교 성도의 신앙 성장을 AI가 함께 돕습니다"

침신 말씀노트는 침례신학대학교(침신대)와의 공식 파트너십을 통해 개발된 **침례교 전용 AI 말씀비서 앱**입니다. 주일 설교를 한 주간 매일 삶으로 연결하여, 성도의 신앙 성장을 돕고 목회자에게 목회 인사이트를 제공합니다.

### 1.3 문제 정의

| 문제 | 현황 | 영향 |
|------|------|------|
| 설교 기억률 저하 | 설교 내용의 90%를 1주일 내 잊음 | 신앙 적용 불가 |
| 주중 신앙 단절 | 주일 예배 후 6일간 신앙 활동 없음 | 신앙 성장 정체 |
| 성장 측정 불가 | 주관적 느낌에만 의존 | 동기부여 부족 |
| 목회 효과 불투명 | 설교 영향력 파악 어려움 | 목회 방향 설정 어려움 |

### 1.4 목표 지표 (OKR)

**Objective**: 침례교 성도의 일상적 신앙 생활 지원

| Key Result | 목표 | 측정 방법 |
|------------|------|----------|
| MAU (월간 활성 사용자) | 파일럿 종료 시 1,000명 | Firebase Analytics |
| 주간 완료율 | 60% 이상 (주간 과제 완료) | Firestore 데이터 |
| 사용자 만족도 | NPS 50 이상 | 앱 내 설문 |
| 교회 도입률 | 1년 내 50개 교회 | 계약 교회 수 |

Goal:

## 3. 기능 요구사항

### 3.1 Phase 1: MVP (2026년 4-6월)

#### 3.1.1 인증 및 교회 연결

**회원가입 플로우**
```
1. 앱 다운로드
2. 교회 코드 입력 (4자리)
3. 기본 정보 입력 (이름, 연락처)
4. 목사 승인 또는 자동 승인
5. 온보딩 완료
```

**기능 상세**

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 교회 코드 연결 | 교회별 고유 코드로 교인 연결 | P0 |
| 소셜 로그인 | 카카오, 구글 로그인 | P0 |
| 프로필 설정 | 이름, 사진, 알림 설정 | P1 |
| 비밀번호 찾기 | 이메일 인증 | P1 |

#### 3.1.2 설교 보기/듣기

**목사/관리자 업로드**
```
업로드 방법:
1. 앱 내 관리자 모드 (목사)
2. 웹 대시보드 업로드
3. YouTube 링크 연동

업로드 정보:
- 설교 제목
- 설교 날짜
- 본문 말씀 (예: 요한복음 3:16)
- 오디오 파일 또는 YouTube URL
- 설교 전체 텍스트 (선택)
```

**AI 자동 처리 (Pastors.ai)**
```
처리 항목:
1. 오디오 → 텍스트 변환 (STT)
2. 핵심 메시지 3가지 추출
3. 요약문 생성 (300자)
4. 주간 묵상 5일 생성
   - 말씀 구절
   - 묵상 질문
   - 적용 방법
5. 처리 시간: 30분 이내
```
