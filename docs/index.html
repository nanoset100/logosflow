앱 소개

말씀브릿지는 주일 설교와 일상을 연결하는 말씀 묵상 앱입니다. 한국 대표 목회자의 설교를 자동으로 요약하고, 월요일부터 금요일까지 5일 묵상 콘텐츠로 변환하여 매일 말씀과 함께하도록 돕습니다.

자주 묻는 질문 FAQ

앱이 실행되지 않거나 오류가 발생해요

앱을 완전히 종료한 후 다시 실행해주세요. 증상이 지속되면 앱을 삭제 후 앱스토어에서 최신 버전으로 재설치해주세요.

설교 요약이 보이지 않아요

설교 요약은 담당자가 설교를 등록한 후 자동으로 생성됩니다. 아직 등록되지 않은 경우 교회 관리자에게 문의해주세요.

묵상 내용이 업데이트되지 않아요

홈 화면을 아래로 당겨 새로고침해주세요. 인터넷 연결 상태를 확인 후 다시 시도해주세요.

기도제목이 저장되지 않아요

로그인 상태를 확인해주세요. 기도제목은 계정에 연결되어 저장됩니다. 로그아웃 상태에서는 저장되지 않습니다.

알림이 오지 않아요

기기 설정에서 말씀브릿지 앱의 알림 권한이 허용되어 있는지 확인해주세요.
설정 > 알림 > 말씀브릿지 > 알림 허용

계정을 삭제하고 싶어요

앱 내 계정 설정에서 계정 삭제를 요청하거나, 아래 문의 이메일로 연락주시면 처리해드립니다. 계정 삭제 시 기도제목 등 모든 데이터가 영구 삭제됩니다.

개인정보 처리방침 Privacy Policy

최종 업데이트: 2025년   최신

1. 수집하는 정보

  • 계정 정보: 이메일 주소 (로그인 시)
  • 기도제목: 사용자가 직접 입력한 기도 내용 및 날짜
  • 앱 사용 기록: 묵상 완료 여부, 활동 통계 (개인 식별 불가능한 형태)

2. 수집하지 않는 정보

  • 이름, 전화번호 등 계정 외 개인 식별 정보
  • 위치 정보
  • 연락처 정보
  • 광고 ID

3. 정보 저장 및 보안

모든 데이터는 Google Firebase의 안전한 클라우드 서버에 암호화되어 저장됩니다. 데이터 전송 시 HTTPS 프로토콜을 사용합니다.

4. 정보 공유

사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다. 광고 목적으로 정보를 사용하지 않습니다.

5. 설교 요약 기능 안내

설교 요약 및 묵상 콘텐츠 생성 기능은 자동화 기술을 활용합니다. 처리 과정에서 설교 텍스트가 일시적으로 서버에 전송되며, 처리 완료 후 저장되지 않습니다.

6. 아동 개인정보 보호

13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 사용자 권리

  • 저장된 기도제목 등 데이터를 언제든지 확인, 수정, 삭제할 수 있습니다.
  • 계정 삭제를 요청할 수 있으며, 삭제 시 모든 데이터가 영구 삭제됩니다.
  • 앱 사용을 언제든지 중단할 수 있습니다.

8. 정책 변경

개인정보 처리방침이 변경되는 경우 이 페이지에서 공지합니다.
