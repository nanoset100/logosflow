# AI 말씀비서 프로젝트 - 완전 개발 문서 세트 작성 요청

## 🎯 프로젝트 개요

### 프로젝트명
**AI 말씀비서 (AI Bible Assistant)**

2가지 버전:
- **침신 말씀노트** - 침례교 전용 (3,500개 교회)
- **말씀브릿지** - 일반 교회용 (전국 교회)

---

## 💡 핵심 컨셉

> "설교가 삶이 되게 하는 AI 비서"

### 문제
- 설교 기억률 10% (주일 설교를 월요일에 잊음)
- 주일만 신자 (주중 신앙 실천 어려움)
- 신앙 성장 측정 불가
- 목회 효과 가시화 어려움

### 솔루션
AI 말씀비서가:
1. 주일 설교를 월~금 매일 상기시킴
2. 구체적 실천 가이드 제공
3. 따뜻한 격려와 응원
4. 성장 가시화 (데이터)
5. 목사님께 목회 인사이트 제공

---

## 🎨 브랜딩

### 침신 말씀노트 (침례교)
- **메인**: 침신 말씀노트
- **영문**: Chimshin Bible Note
- **부제**: AI 말씀비서와 함께하는 신앙 성장
- **포지셔닝**: 침신대 공식 앱
- **타겟**: 침례교 3,500개 교회
- **가격**: 교인당 월 250원

### 말씀브릿지 (일반)
- **메인**: 말씀브릿지
- **영문**: WordBridge
- **부제**: AI 말씀비서가 설교를 삶으로 연결합니다
- **포지셔닝**: 설교와 삶을 잇는 다리
- **타겟**: 전국 모든 교회
- **가격**: 교인당 월 300원

## 📱 핵심 기능 (3단계 출시)

### Phase 1: MVP (2026.4-6월 파일럿)
**"어른들도 쉽게!"** 필수 기능만:

#### 설교 보기/듣기
- 주일 설교 업로드 (목사/관리자)
- AI 자동 요약 (Pastors.ai)
- 핵심 메시지 추출

#### 주간 적용 (월~금)
| 요일 | 테마 | 내용 |
|------|------|------|
| 월 | 깨달음 | 설교 복습 |
| 화 | 계획 | 실천 계획 |
| 수 | 점검 | 중간 체크 |
| 목 | 심화 | 더 깊은 적용 |
| 금 | 정리 | 주간 회고 |

#### AI 격려 시스템 ⭐⭐⭐
- 상황별 맞춤 격려
- 따뜻한 톤
- 연속 달성 축하
- 쉬었을 때 부드러운 격려

#### 푸시 알림 ⭐⭐⭐
- 아침 9시: "오늘의 말씀 준비됐어요"
- 점심 12시: "짧은 묵상"
- 저녁 8시: "하루 마무리"

#### 간단한 기록
- 주간 완료 현황
- 간단한 메모
- 체크 기능

#### Phase 1 제외 항목
- ❌ 레벨/경험치
- ❌ 배지 시스템
- ❌ 교회 랭킹
- ❌ 복잡한 그래프
- ❌ 도전 과제

### Phase 2: 커뮤니티 (2026.9-12월)
**"혼자가 아니라 함께"**

#### 간증 나눔 ⭐⭐⭐
- 간단한 간증 작성
- 댓글/좋아요
- 목사님 확인 가능

#### 기도 제목 ⭐⭐⭐
- 기도 제목 올리기
- 함께 기도하기
- 긴급/지속 구분

#### 격려 기능
- 간단한 댓글
- 좋아요 버튼

### Phase 3: 성장 가시화 (2027년~)
**"내가 이렇게 자랐구나"**

#### 간단한 통계
- 3개월 성장 그래프
- 완료율 추이
- 카테고리별 비교

#### 배지 (소수만!)
- 5-10개 정도만
- 의미 있는 것만
- 과하지 않게

```

- iOS + Android 동시
- 네이티브급 성능

상태관리: Riverpod
디자인: Material Design 3

### Backend
```
초기: Firebase
- Authentication
- Firestore Database
- Cloud Functions
- Storage
- Cloud Messaging (푸시)

확장 시: Supabase 고려
```

### AI 처리
```
Phase 1 (파일럿):
- Pastors.ai API
- Team 플랜 ($75/월)
- 수동 업로드 (우리가 직접)

Phase 2 (확산):
- GPT-4 자체 시스템
- Whisper (음성→텍스트)
- GPT-4 (요약 생성)
- 비용: 설교당 $0.5
- 완전 자동화
```

## 🎨 UI/UX 원칙 (어른 친화적!)

### 디자인 원칙

1. **큰 글씨, 큰 버튼**
   - 최소 18pt 이상
   - 터치 영역 44pt 이상

2. **명확한 네비게이션**
   - 3-4개 탭만
   - 단순한 구조

3. **따뜻한 색상**
   - Primary: 파란색 `#4A90E2`
   - Secondary: 초록색 `#7ED321`
   - Accent: 금색 `#F5A623`

4. **친근한 메시지**
   - "~하세요" (존댓말)
   - 이모지 적절히
   - 따뜻한 톤

## 💰 비즈니스 모델

### 수익 구조

**침례교 (침신 말씀노트):**
- 교인당 월 250원
- 수익 분배:
  - 운영 40% (개발, AI, 서버)
  - 침신대 30% (파트너십, 브랜드)
  - 개발사 30% (유지보수, 개선)

**일반 (말씀브릿지):**
- 교인당 월 300원
- 수익 분배:
  - 운영 50%
  - 개발사 50%

### 예상 수익 (침례교 50% 침투)
```
1,750개 교회 × 평균 150명 = 262,500명

월 수익: 262,500명 × 250원 = 6,562만원
연 수익: 7억 8,750만원
침신대 몫 (30%): 연 2억 3,625만원
