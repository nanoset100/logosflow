앱이 무엇을 해주나요?

매주 설교를 놓치지 않고, 말씀과 함께하는 일상을 만들어 드립니다.

🔐

간편 로그인

이메일과 비밀번호로 쉽게 회원가입하고, 한 번 로그인하면 다음에는 자동으로 접속됩니다.

  • 이메일로 회원가입 / 로그인
  • 교회 코드 입력으로 내 교회 연결
  • 자동 로그인 유지
📝

설교 목록 & 상세 보기

목사님이 등록한 설교를 언제든지 찾아볼 수 있습니다. 제목, 본문, 날짜별로 정리되어 있어요.

  • 최신 설교 목록 자동 업데이트
  • 설교 본문, 제목, 날짜 표시
  • AI가 정리한 설교 핵심 요약 제공
🤖

AI 설교 요약

긴 설교 내용을 AI가 핵심만 골라서 3~4문단으로 요약해 드립니다. 바쁜 성도님도 설교 내용을 쉽게 복습할 수 있어요.

  • 전체 요약: 3~4문단 핵심 정리
  • 핵심 교훈: 글머리 목록으로 정리
  • 주중 묵상 가이드 자동 생성
🔊

설교 음성 듣기 (TTS)

설교 요약 내용을 AI 음성으로 들을 수 있습니다. 운전 중이나 집안일 할 때도 말씀을 들어보세요!

  • 텍스트를 자연스러운 음성으로 변환
  • 재생 / 일시정지 / 진행 슬라이더
  • 한국어 음성 지원
💾

설교 저장하기

마음에 드는 설교를 저장해 두면, 인터넷 없이도 언제든지 다시 읽을 수 있습니다.

  • 설교 북마크 기능
  • 저장된 설교 모아보기
  • 오프라인에서도 열람 가능
🙏

나의 기도제목

기도제목을 앱에 기록하고 관리하세요. 응답받은 기도는 완료 체크로 표시할 수 있어요.

  • 기도제목 작성 & 목록 관리
  • 응답 완료 체크 기능
  • 밀어서 삭제 (스와이프)
🌅

주중 묵상 가이드

설교를 한 번 듣고 끝이 아닙니다. 월~금 5일 동안 매일 다른 묵상 포인트를 안내해 드려요.

  • 월~금 5일 묵상 내용 자동 생성
  • 날짜별 진행 현황 표시
  • 묵상 완료 체크 기능
👥

구역 예배 (소그룹)

구역 모임에서 함께 설교를 나눌 수 있도록 구역 예배 전용 화면을 제공합니다.

  • 구역 예배용 설교 요약 화면
  • 5일 묵상 나눔 가이드
  • 카카오톡으로 설교 내용 공유
📊

이번 주 활동 현황

내가 얼마나 말씀을 가까이했는지 한눈에 볼 수 있어요. 꾸준한 신앙 생활을 응원합니다!

  • 예배 / 묵상 / 성경 읽기 통계
  • 연속 출석 일수 표시
  • 주간 활동 그래프
📖

오늘의 성경

신약 260장을 1년 동안 매일 읽는 계획표입니다. 오늘 읽을 성경 본문을 앱이 안내해 드려요.

  • 신약 260장 1년 통독 계획
  • 오늘 읽을 본문 자동 표시
  • 왕초보 성경통독 앱 연동

누가 사용하나요?

일반 성도와 관리자(목사님/간사님)가 각각 다른 기능을 사용합니다.

👤 일반 성도

  • 설교 목록 보기 & 검색
  • AI 요약 및 음성 듣기
  • 설교 저장 및 북마크
  • 주중 묵상 가이드 따라가기
  • 기도제목 작성 및 관리
  • 구역 예배 나눔 참여
  • 이번 주 활동 현황 확인
  • 카카오톡으로 설교 공유

🛠️ 관리자 (목사님 / 간사님)

  • 설교 직접 입력 등록
  • 유튜브 링크로 설교 등록
  • 음성 파일 업로드로 자동 변환
  • AI 자동 요약 생성 실행
  • 설교 수정 및 삭제
  • 성도 목록 관리

어떻게 사용하나요?

처음 사용하는 분도 5분이면 시작할 수 있어요!
