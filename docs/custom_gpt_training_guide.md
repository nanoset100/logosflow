# 우리만의 맞춤형 설교 요약 AI (Custom GPT) 만드는 방법

Pastors.ai에서 얻은 결과물(정답지)을 활용해 **"우리 앱 전용 맞춤형 AI"**를 만드는 과정을 **'파인튜닝(Fine-tuning, 미세 조정)'**이라고 부릅니다. 

쉽게 비유하자면, 아주 똑똑하지만 세상 모든 지식을 다 알고 있어서 답변이 중구난방인 대학생(GPT-4o)에게, **"너는 이제부터 우리 교회 부목사님처럼, 이 500개의 참고서(Pastors.ai 요약본) 스타일로만 대답해야 해!"** 라며 족집게 과외를 시키는 과정입니다.

이 과정을 4단계로 아주 쉽게 설명해 드립니다.

---

## 1단계: 황금 데이터(학습지) 모으기 🌟 (가장 중요)

AI를 똑똑하게 만들려면 "좋은 문제"와 "만점짜리 정답"의 쌍이 필요합니다.
엑셀(Excel)이나 구글 스프레드시트에 아래와 같이 표를 만들어 차곡차곡 모아주세요. (최소 100개, 권장 300~500개 이상)

| 번호 | 시스템 역할 (System) | 입력 데이터: 원문 (User) | 출력 데이터: 만점 정답 (Assistant) |
|---|---|---|---|
| 1 | 너는 탁월한 목상 가이드야. | [이찬수 목사님 설교 유튜브 자막 전체 텍스트...] | [Pastors.ai가 만들어준 5일치 묵상 요약본 텍스트...] |
| 2 | 너는 탁월한 목상 가이드야. | [오정현 목사님 설교 유튜브 자막 전체 텍스트...] | [Pastors.ai가 만들어준 일일 묵상 텍스트...] |
| ... | ... | ... | ... |

* 💡 **팁:** 정답(Assistant) 데이터는 우리가 최종적으로 유저에게 보여주고 싶은 완벽한 형태(예: JSON 포맷, 줄바꿈 등)로 다듬어서 저장해야 합니다. AI는 이 형태까지 똑같이 따라 배웁니다.

## 2단계: AI가 먹을 수 있는 형태(JSONL)로 변환하기 🔄

엑셀 파일은 사람이 보기 좋지만, OpenAI는 이 파일을 **JSONL**이라는 특수한 컴퓨터 언어 문서로 바꿔달라고 요구합니다. (파이썬 코드 한 줄이면 자동으로 엑셀을 이 형태로 변환할 수 있습니다)

**JSONL 파일의 한 줄 예시:**
```json
{"messages": [{"role": "system", "content": "너는 탁월한 묵상 가이드야."}, {"role": "user", "content": "성도 여러분 오늘 본문은... (설교 스크립트)"}, {"role": "assistant", "content": "【오늘의 묵상】\n1일차: ..."}]}
```
* 이렇게 생긴 데이터가 500줄 들어있는 하나의 파일(`training_data.jsonl`)을 만듭니다.

## 3단계: OpenAI에 과외 선생님(데이터) 던져주기 🚀

개발을 전혀 몰라도 OpenAI 홈페이지(웹사이트)에서 마우스 클릭 몇 번으로 과외를 시킬 수 있습니다.

1. **OpenAI 플랫폼 접속:** [platform.openai.com/finetune](https://platform.openai.com/finetune) 로 접속합니다.
2. **새 모델 만들기 클릭:** 우측 상단의 `Create new 모델` 버튼을 누릅니다.
3. **베이스 모델 선택:** 우리는 똑똑하면서도 가격이 아주 저렴한 **`gpt-4o-mini`** 모델을 선택합니다. (가성비가 최고입니다)
4. **데이터 업로드:** 2단계에서 만든 `training_data.jsonl` 파일을 업로드합니다.
5. **학습 시작 (Train):** 버튼을 누르고 커피 한 잔 마시며 기다립니다. (데이터 500개 기준 보통 1~2시간 안에 끝납니다)

## 4단계: 나만의 고유한 AI 모델 탄생 및 사용 🎉

학습이 성공적으로 완료되면, 세상에 하나밖에 없는 **우리만의 AI 모델 이름표(ID)**가 생깁니다.
* 예: `ft:gpt-4o-mini-2024-07-18:my-church-org:wordbridge-v1:8abcDeF`

이제 우리 앱(말씀브릿지)의 백엔드 서버에서 원래 부르던 `gpt-4o` 대신에, **저 길다란 새 모델 아이디를 호출**하기만 하면 됩니다.

### 🌟 파인튜닝(맞춤형 학습) 완료 후 나타나는 기적

1. **프롬프트를 길게 안 써도 됩니다:** 기존에는 "너는 목사님이야. 1일차는 이렇게 쓰고 2일차는 저렇게 쓰고, 말투는 다정하게 해줘..." 라고 수천 자의 명령을 써야 했지만, 이제는 "이 설교 요약해" 라고만 던져도 **알아서 Pastors.ai 퀄리티로 5일치 묵상본을 척척 뱉어냅니다.** (명령어가 짧아지니 API 호출 비용이 크게 절감됩니다)
2. **결과물이 튀지 않습니다:** AI가 헛소리를 하거나 말투가 로봇처럼 변하는(할루시네이션) 현상이 거의 사라지고, 딱 우리가 넣어준 학습 데이터 스타일로만 안정적으로 답변합니다.

---

**결론:** 
초반 몇 달간 관리자가 조금 고생해서 엑셀에 300~500개 정도의 "설교 원문 + Pastors.ai 결과물" 세트만 정성스럽게 긁어 모아주시면, 그 다음부터는 개발자가 클릭 몇 번으로 우리만의 똑똑한 묵상 AI를 찍어낼 수 있습니다!
