# Wireframes: AI 말씀비서
## 화면 설계 문서 v1.0

---

## 개요

### 디자인 원칙
```
어른 친화적 UI 핵심 원칙:
1. 큰 글씨: 최소 18pt, 제목 24pt 이상
2. 큰 버튼: 최소 높이 56px, 터치 영역 44pt
3. 단순 구조: 화면당 핵심 행동 1가지
4. 명확한 계층: 무엇을 해야 하는지 즉시 파악
5. 따뜻한 색상: 차갑지 않은 블루, 그린 계열
6. 여백 활용: 과밀하지 않은 레이아웃
```

### 화면 목록 (10개)
```
1.  Splash Screen (스플래시)
2.  Onboarding (온보딩 3단계)
3.  Login/Register (로그인/회원가입)
4.  Home (홈 - 메인)
5.  Sermon Detail (설교 요약 상세)
6.  Daily Devotion (주간 적용 - 일일)
7.  My Record (나의 기록)
8.  Testimony (간증 나눔 - Phase 2)
9.  Prayer (기도 제목 - Phase 2)
10. Pastor Dashboard (목사 대시보드)
```

## 13. 접근성 체크리스트

```
텍스트:
□ 최소 폰트 크기 18pt 준수
□ 행간 1.5 이상
□ 색상만으로 정보 전달하지 않음
□ 명도 대비 4.5:1 이상 (WCAG AA)

터치:
□ 최소 터치 영역 44×44pt
□ 버튼 간 최소 8px 간격
□ 실수로 탭 방지 (취소 가능)

내비게이션:
□ 항상 "뒤로가기" 제공
□ 현재 위치 표시
□ 탈출구 명확 (홈 버튼)

미디어:
□ 이미지 alt 텍스트
□ 오디오 자막/텍스트 버전
□ 동영상 자막

스크린 리더:
□ 모든 UI 요소 라벨링
□ 탭 순서 논리적
□ 포커스 표시 명확
```
