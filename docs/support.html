자주 묻는 질문 (FAQ)

  • 교회 코드를 모릅니다.
    담당 목사님 또는 교회 관리자에게 문의해 주세요.
  • 로그인이 안 됩니다.
    이메일/비밀번호를 확인하시고, 비밀번호를 잊으셨다면 로그인 화면의 "비밀번호 찾기"를 이용해 주세요.
  • 설교 오디오가 재생되지 않습니다.
    인터넷 연결 상태를 확인하고 앱을 재시작해 주세요.
  • 기도제목이 저장되지 않습니다.
    로그인 상태에서만 저장됩니다. 로그인 여부를 확인해 주세요.
  • 앱 삭제 후 데이터가 사라졌습니다.
    기도제목은 서버에 저장되므로 재로그인 시 복구됩니다. 로컬 활동 기록은 복구되지 않습니다.
